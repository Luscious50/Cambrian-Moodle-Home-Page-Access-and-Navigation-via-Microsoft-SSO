# Manual Test Checklist

## Objective
Support the automated smoke checks with manual verification evidence.

## Manual checks
- Open Moodle login page
- Confirm page loads without error
- Confirm Microsoft SSO sign-in option is visible
- Log in with valid student account
- Confirm Moodle home page loads
- Confirm visible homepage navigation items are accessible
- Confirm important links/buttons are present
- Capture screenshots for each successful step

## Evidence to capture
- Moodle login page
- Microsoft sign-in screen
- Post-login Moodle homepage
- At least one navigation action from homepage
