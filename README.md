## Project Summary
This repository contains the technical artifacts for the CMP1979 capstone testing project on **Cambrian Moodle Home Page Access and Navigation**. The project focuses on the student-facing Moodle access flow and the main homepage navigation elements available after login.

The testing scope includes:
- Moodle login page access
- Microsoft SSO authentication flow
- Homepage primary navigation
- Homepage visible links and controls

This repository is used to store automation files, supporting evidence, and project-related testing artifacts.

## Repository Structure
- `/tests` – automated tests
- `/evidence` – screenshots and supporting evidence
- `/models` – diagrams and models
- `/.github/workflows` – automated quality checks
- `README.md` – project overview and setup information

## Setup
1. Clone this repository.
2. Open the project in your preferred editor.
3. Review the `/tests` and `/evidence` folders.
4. Install any required dependencies for the selected testing tools.
5. Run the automated tests using the command or framework selected by the group.
